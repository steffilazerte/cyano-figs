Figures for Cyanobook

This is the first pass of figures for the Cyano book. Figures are created individually and then combined into multi-part figures.

For the largest figure, I show the individual panels one-by-one along with questions. Then at the end I put them together.

Setup the data and the base plots

Code 
fig2_4 <- cyano |>
  mutate(humic_fct = if_else(humic == 1, "Coloured", "Clear"))

tp_lim <- c(10, 30) # Keep 100?
y_breaks <- c(0.1, 1, 10, 100, 1000)

# y-Log base plot
g0 <- ggplot(fig2_4, aes(x = tp_epi)) +
  theme_classic() +
  theme(plot.margin = margin()) +
  scale_x_log10(name = bquote('TP'~(mu*g/L)), breaks = c(1, 10, 100, 1000)) +
  scale_y_log10(breaks = y_breaks) +
  annotation_logticks()

# Non-y-log base plot
g0.1 <- ggplot(fig2_4, aes(x = tp_epi)) +
  theme_classic() +
  theme(plot.margin = margin()) +
  scale_x_log10(name = bquote('TP'~(mu*g/L)), limits = c(5, 50), breaks = c(1, 10, 20, 30, 40, 50)) +
  annotation_logticks(sides = "b")
